import { Aside, LinkCard, CardGrid, Card, Steps } from '@astrojs/starlight/components';

<Steps>
1. Перейдите на [официальный сайт Supabase](https://supabase.com/)
2. Зарегистрируйтесь:
    - если на вашем устройстве настроен `GitHub`, выберите `Continue with GitHub`, 
    - иначе - перейдите по ссылке `Sign Up Now` и зарегистрируйтесь и подтвердите email
3. Войдя в аккаунт, выберите `New Project`
4. Создайте проект:
    - введите имя проекта, 
    - придумайте (и запишите) пароль к базе данных,
    - регион можно не менять
5. Дождитесь завершения процесса создания, а затем скопируйте значения полей `Project URL` и `API Key`, 
они неоднократно понадобятся вам и фронтенду 
</Steps>

<Aside type="note" title="Рекомендация">
    Возможно, будет удобно записать `URL` и `API KEY` в одну из карточек в таск-трекере
</Aside>
